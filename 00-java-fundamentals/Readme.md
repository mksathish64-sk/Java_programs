# Java Fundamentals

This folder contains the core concepts required to understand how Java works before writing Java programs.

## Topics Covered

1. Why Java
2. JDK, JRE, JVM
3. Java Compilation Process
4. Java Program Execution Process
5. Java Program Structure
6. Main Method Keywords
7. First Java Program

## Learning Outcome

After completing this section, you will understand:

- Why Java is platform-independent.
- The roles of JDK, JRE, and JVM.
- How Java programs are compiled and executed.
- The structure of a Java program.
- How the JVM starts program execution.
