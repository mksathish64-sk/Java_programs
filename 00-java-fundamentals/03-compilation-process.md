# Java Compilation Process

## What is Compilation?

Compilation is the process of converting Java source code (`.java`) into Java bytecode (`.class`).

## How does Compilation Work?

1. Write the Java source code in a `.java` file.
2. Use the Java compiler (`javac`) available in the JDK.
3. Run the following command:

```bash
javac FileName.java
```

        +--------------+                                  +----------+
        |              |          JDK (javac)             |          |
        | source code  |  ------------------------------> | Bytecode |
        |   (.java)    |          Compilation             | (.class) |
        +--------------+                                  +----------+

4. The compiler checks the source code for syntax and compilation errors.
5. If no errors are found, the compiler generates platform-independent bytecode (`.class` file).
6. A separate `.class` file is created for each class defined in the source file.

## Output

- **Input:** `FileName.java`
- **Output:** `FileName.class` (Bytecode)

> The generated bytecode is platform-independent and can be executed by the Java Virtual Machine (JVM).

## Key Takeaways

- The Java compiler (`javac`) converts source code into bytecode.
- Compilation checks for syntax and semantic errors.
- A `.class` file is generated for each compiled class.
- Bytecode is platform-independent.
